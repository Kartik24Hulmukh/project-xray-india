# Security policy

Report vulnerabilities privately to the address configured by the deploying organization. Do not open public issues containing exploit details, personal information or unpublished evidence.

Supported version: latest tagged release. Target acknowledgement: 2 business days. Critical mitigation target: 72 hours. Rotate exposed credentials immediately. See `docs/THREAT_MODEL.md` and `docs/INCIDENT_RESPONSE.md`.


For hosted preview incidents, follow `docs/ops/KILL_SWITCH_RUNBOOK.md` and preserve audit evidence.
