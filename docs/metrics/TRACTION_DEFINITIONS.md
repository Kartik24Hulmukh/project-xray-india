# Traction and impact metric definitions

These are definitions only. No fabricated counts are included.

## North star for preview

**Independent, policy-compliant synthetic dossiers completed without live coaching.**

## Preview metrics

| Metric | Definition | Good signal |
|---|---|---|
| External quickstart attempts | Distinct external people who start the workflow | ≥5 in 24h |
| Successful runs | Completed create→review→publish→verify path | ≥3 |
| Independent reproductions | Completions without live maintainer coaching | ≥1–2 |
| Substantive issues | Actionable bugs/docs/safety reports | ≥3 |
| Domain-expert conversations | Qualified research/editorial conversations | ≥2 |
| Security/privacy incidents | Real leak, auth bypass, or real-data exposure | **0** |

## Non-headline metrics

Stars, likes, impressions, waitlist size, page views.

## 30-day traction (after preview)

- three professional teams evaluating
- two completed real or historical workflows under governance
- one repeat workflow
- preparation-time reduction measured
- exact-source acceptance high
- zero unsupported publication
- one serious pilot discussion or funded pilot

## Impact ladder

`dossier used → document requested → response received → record corrected/disclosed → formal review → repair/recovery/policy change`

Do not claim impact rungs that have not occurred.
