# Design-Partner Scorecard

Per partner, record:
- Baseline dossier time (their current workflow, self-reported)
- Time to first anchored claim using Tracepaper
- Reviewer reconstruction time (second reviewer re-derives a claim from anchors)
- Missing-anchor rate across a sanitized document set
- Evidence gaps identified that their current workflow missed
- Repeat-use commitment (yes/no + planned cadence)
- Paid-pilot condition (the single thing that would unlock payment)

Wednesday-week targets: 20 personalized messages, 5 workflow interviews,
2 sanitized document tests, 1 signed design-partner commitment, 1 paid-pilot condition.
