# Design-partner interview guide

## Goals
Understand time-to-value for evidence dossiers, not vanity metrics.

## Questions
1. What investigation did you try first?
2. Time from login to first project?
3. Time to first anchored claim?
4. Time to first dual-reviewed claim?
5. Time to first published dossier?
6. What blocked you (auth, storage, review, UI)?
7. Would you run a second dossier next week?
8. What decision did this dossier support?
9. Privacy concerns with evidence handling?
10. Likelihood to adopt (1–5) and why?
