# Reviewer workflow guide

1. Open only claims assigned for review (correct version).
2. Verify source class, passage anchor, and quarantine state is clean.
3. Approve or reject with rationale in the review decision field.
4. Never approve your own claim version.
5. Second reviewer must be a distinct principal.
6. After dual approval, admin may publish that exact version.
7. Corrections require a new version and a fresh two-person review.
