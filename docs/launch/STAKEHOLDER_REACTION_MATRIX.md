# Stakeholder reaction matrix — India launch scenarios

This matrix is a premortem and response guide for a **synthetic technical preview**. It is not legal advice.

| Stakeholder | Honest preview reaction | Failure mode if overclaimed | Response posture |
|---|---|---|---|
| Investigative journalists | Useful if it shortens evidence assembly | “Truth engine / scoop bait” | Demo provenance + corrections; invite workflow feedback only |
| RTI / civic researchers | Interest in gap → request drafts | Expect live cases and authority leverage | Emphasize draft tooling; no case sponsorship |
| Data journalists / OSINT | Respect for anchors and capsules | Distrust if labels conflict | Freeze SHA; clean-clone proof |
| Civil servants / PSU engineers | Cautious curiosity | Fear of gotcha tool | Offer response channel framing; no scores |
| Contractors / bidders | Defensive | Defamation / commercial harm fears | Synthetic only; no real contracts |
| Politicians / parties | Ignore if small; attack if framed as anti-corruption AI | “Political weapon” narrative | Refuse partisan framing; no candidate ranking |
| Courts / legal observers | Demand precise language | Contempt risk from “verified guilt” language | Document-state language only |
| Platforms (GitHub, hosts, social) | Fine if synthetic + ToS clean | Takedown if real PII/defamation | Kill switch + synthetic markers |
| Security community | Respect fail-closed gates | “Security theater” if MFA/scan unproven | Publish partial ledger; invite adversarial review |
| Open-source engineers | Value tests and capsules | Distrust version/label inconsistency | Honest readiness YAML |

| Hosting providers / AWS | Normal SaaS risk | Abuse complaints | Invite-only + kill switch |

## Sector scenario drills

### If a journalist asks “Does this prove corruption?”
Answer: No. It shows what selected records establish, where they conflict, and what remains unproven after human review.

### If a political actor claims the tool is targeted
Answer: Preview uses synthetic data only. Real publication requires two independent reviewers, source anchors, and operator legal/editorial gates. No candidate ranking exists.

### If a contractor claims defamation risk
Answer: No real contracts or firms are published in this preview. Corrections and withdrawals are first-class for any future reviewed publication.

### If GitHub users expect production SaaS
Answer: This is a reproducible reference implementation and synthetic evaluation package. Operator production gates remain open in `ops/production-readiness.yaml`.
